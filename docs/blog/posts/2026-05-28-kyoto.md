# 五月的京都

游客大多在三四月涌来看樱花，五月反而清净了许多。

<!-- more -->

沿着哲学之道走，绿叶已经很茂盛了，偶尔能看到几朵晚樱还挂在枝头。河边的猫不怕人，懒洋洋地晒太阳。

![京都小路](https://images.unsplash.com/photo-1528360983277-13d401cdc186?w=1200&q=85)

## 推荐的几个地方

南禅寺的水路阁在早上人很少，适合慢慢拍照。下午去锦市场买东西吃，烤扇贝和玉子烧都很好。

晚上在木屋町的居酒屋坐了很久，喝了几杯清酒，听旁边桌的老爷爷讲故事。

!!! tip "出发建议"
    五月比四月便宜很多，酒店和机票都更好订。气温也正好，不冷不热。
