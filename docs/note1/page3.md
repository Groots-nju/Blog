# 海边黄昏

> 有些颜色，调色板里根本没有。

![海边夕阳](https://images.unsplash.com/photo-1507525428034-b723cf961d3e?w=1200&q=85)

## 为什么黄昏的光最好

太阳低角度入射，光线穿过更厚的大气层，短波蓝光散射殆尽，只剩橙红暖调。这是物理，也是诗意。

摄影师把这段时间叫做"黄金时刻"（Golden Hour），大约在日落前后各一小时。

## 海浪声

海浪的声音之所以令人放松，是因为它的频率接近白噪音，能屏蔽其他干扰，帮助大脑进入低唤醒状态。

这就是为什么很多人在海边容易睡着。

![海浪](https://images.unsplash.com/photo-1505118380757-91f5f5632de0?w=1200&q=85)

## 我喜欢在海边做的事

1. 捡石头，专找形状有意思的
2. 坐着发呆，不看手机
3. 等天色从橙变紫再变蓝黑

!!! tip "小建议"
    去海边不要带太多东西，一条毯子、一瓶水就够了。越轻松，越容易感受到什么。
