# 山间晨雾

> 那些早起的人，才能看到世界最温柔的样子。

清晨五点，山谷还笼罩在一片寂静里。空气带着湿润的草木气息，脚下的泥土踩上去是软的。

![山间晨雾](https://images.unsplash.com/photo-1598135753163-6167c1a1ad65?w=1200&q=85)

## 为什么要早起

早起不是一种自律，是一种对光线的珍惜。那段时间的光是金色的、斜的，能把最普通的树叶照成油画。

等到八点，光就变白了，什么都平平无奇。

## 晨雾的形成

薄雾是因为夜间地面辐射降温，近地面水汽凝结所致。温度越低、湿度越高，雾越厚。所以山间秋冬最多雾，夏天反而少。

![林间小路](https://images.unsplash.com/photo-1448375240586-882707db888b?w=1200&q=85)

## 一些碎片

- 带一件厚外套，山里的早晨比你想象的冷
- 不要带耳机，听风声和鸟叫
- 手机放在包里，先用眼睛记

!!! tip "小提示"
    如果想拍好晨雾照片，建议使用手动对焦，因为自动对焦在低对比度场景里容易迷失。
