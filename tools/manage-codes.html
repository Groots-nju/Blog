🔐 邀请码管理工具

在本地运行，不上传任何数据

① 生成新邀请码

数字填 1 生成单个，填 5 一次生成5个随机码。

② 邀请码列表

还没有邀请码，点上方「生成」按钮创建。

③ 复制哈希到 gate.js

复制后打开 docs/javascripts/gate.js，替换 var GATE_CODES = [...] 里的内容，然后重新发布。

📋 使用步骤

在上方生成邀请码，把明文码（绿色）发给朋友
点「复制全部哈希」，打开 docs/javascripts/gate.js
找到 var GATE_CODES = [ 那行，把复制的内容粘贴进去替换
终端运行 git add . && git commit -m "update codes" && git push && mkdocs gh-deploy
把明文邀请码发给朋友，他们输入后就能进入网站
撤销某人：点「撤销」按钮，重新复制哈希并发布即可
